In order to install LibreOffice onto Windows PC, it is necessary to download the installation file.

Pre-requisites

  • You will need an Internet connection in order to download a necessary file.

Steps

  1. Type LibreOffice into the search bar in your browser (Figure 1).
  2. Locate the official LibreOffice website and click download. The file will take several minutes to download. (Figure 2) and (Figure 3)

    3. *It will open a window asking where to save if such option is set. If not, then look for the file in a default Download folder.

  3. Double Click on LibreOffice_4.3.0*.msi. It will start the installation process.
  4. Follow the on-screen prompts.

    5. *If the installation is installed successfully, you will see a window as on (Figure 4)
