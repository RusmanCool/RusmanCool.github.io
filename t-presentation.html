Creating a presentation in LibreOffice suite

Steps

If you need to create a good looking presentation, LibreOffice Impress Presentation suite can create a presentation that you will be proud of.


Follow these steps to create a presentation.
  1. Launch either the main LibreOffice window (Figure 1) and choose Impress Presentation (Figure 1).

    2. Or Launch Impress Presentation from the Program Menu (Figure 2)

  2. Create your slides one-by-one in the main window.

    3. In the main Impress window (Figure 3), you can choose slides types, how to combine them, choose how different lines look like, change slides background area, insert different shapes, and many more.

  3. To save the file, click on "File" â€“ "Save as" and choose the suitable format (Figure 4)

    4. *Choose .pptx type if you are planning to open the file in MS Office PowerPoint

  4. After the initial save, you can just use "CTRL + S" key combinations to save the work in progress.
