Creating a spreadsheet in LibreOffice suite

Steps

If you need to create a spreadsheet that is functional and looks good, LibreOffice Calc suite can do just that. Follow these steps to create a spreadsheet.

  1. Launch either the main LibreOffice window (Figure 1) and choose Calc Spreadsheet (Figure 1).

    2. Or Launch Calc Spreadsheet from the Program Menu (Figure 2)

  2. Enter your formulas, numbers, and text into any cells.

    3. In the main Calc window (Figure 3), you can format the font and numbers and use Function Wizard to insert special math function.

  3. To save the file, click on "File" â€“ "Save as" and choose the suitable format (Figure 4)

    4. *Choose .xlsx type if you are planning to open the file in MS Office Excel

  4. After the initial save, you can just use "CTRL + S" key combinations to save the work in progress.
